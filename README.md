# express-ejs-demo

A sample Express app that uses EJS template engine

## How to run this project?

Clone or download the repository. Install dependencies:

```bash
npm install
# --- or ---
yarn install
```

Run the app:
```bash
npm start
# --- or ---
yarn start
```

Visit `localhost:3000` from your web browser.
